malloc vs calloc

Feature malloc calloc
Purpose Allocates a single block of memory Allocates multiple blocks of memory and initializes to zero
Syntax void *malloc(size_t size); void *calloc(size_t num, size_t size);
Initialization Does not initialize memory (garbage values) Initializes memory to zero
Usage Suitable for single block allocation Suitable for array or structure allocation with initialization
Arguments Takes one argument (size) Takes two arguments (num, size)
Example int *ptr = (int *)malloc(sizeof(int)); int *arr = (int *)calloc(5, sizeof(int));
